Требования

Мобильное устройство с установленной Java2ME (Java Micro Edition).
Требования к платформе: MIDP 2.0, CLDC 1.0

Общая информация

Приложение на Java для мобильных устройств.
Программа предназначена для просмотра справочной информации по регионам РФ.
Выбор осуществляется или непосредственным вводом номера или пролистыванием по порядку.
В базе присутствуют номера с 1 по 89.

Управление

Список регионов.
Для просмотра регионов вводите соответствующий номер с клавиатуры или пролистывайте с помощью джойстика.
Для просмотра справочной информации по региону нажмите джойстик.

Справочная информация по региону.
Пролистывайте с помощью джойстика.
Влево/вправо - постранично.
Вверх/вниз - построчно.
Для возврата к списку нажмите джойстик.
# - сброс
* - смена шрифта
